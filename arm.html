Argumentation Metamodel Arguments

The argumentation metamodel (ARM) is an OMG-provided standard that CertWare uses as the basis for more specific argument models. CertWare's GSN, CAE, and EUROCONTROL argument models are built upon the ARM. The analyst can also write an argument model directly in ARM. CertWare provides only a text-based structured tree editor owing to little containment usage in the metamodel.
